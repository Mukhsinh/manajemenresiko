# Panduan Penggunaan Halaman Residual Risk

## Akses Halaman Residual Risk

1. **Login ke Aplikasi**
   - Masukkan email dan password
   - Klik tombol "Login"

2. **Navigasi ke Halaman Residual Risk**
   - Dari sidebar menu, klik "Residual Risk"
   - Halaman akan menampilkan dashboard analisis risiko residual

## Fitur-Fitur Halaman Residual Risk

### 1. Dashboard Statistik
Menampilkan 4 kartu statistik utama:
- **Total Residual Risks**: Jumlah total risiko residual
- **Avg Inherent Value**: Rata-rata nilai risiko inherent
- **Avg Residual Value**: Rata-rata nilai risiko residual
- **Risk Reduction**: Persentase pengurangan risiko

### 2. Filter Data
Gunakan dropdown filter untuk menyaring data:
- **Rencana Strategis**: Filter berdasarkan rencana strategis
- **Unit Kerja**: Filter berdasarkan unit kerja
- **Kategori Risiko**: Filter berdasarkan kategori risiko

### 3. Visualisasi Chart

#### Residual Risk Matrix
- Scatter plot yang menunjukkan posisi risiko berdasarkan probabilitas dan dampak
- Setiap titik mewakili satu risiko residual
- Warna titik menunjukkan level risiko (hijau=rendah, kuning=sedang, merah=tinggi)

#### Inherent vs Residual Comparison
- Bar chart perbandingan nilai risiko inherent vs residual
- Menunjukkan efektivitas penanganan risiko
- Semakin besar selisih, semakin efektif penanganan risiko

### 4. Tabel Detail Data
Menampilkan informasi lengkap setiap risiko residual:
- **Risk Code**: Kode unik risiko
- **Unit Kerja**: Unit kerja yang bertanggung jawab
- **Inherent**: Nilai risiko sebelum penanganan
- **Residual**: Nilai risiko setelah penanganan
- **Reduction**: Persentase pengurangan risiko
- **Level**: Tingkat risiko (Low, Medium, High, Extreme)
- **Review Status**: Status review risiko
- **Next Review**: Tanggal review berikutnya

## Fitur Download Laporan

Klik tombol "Download" untuk mengakses berbagai format laporan:

### 1. Excel Report
- **Format**: File .xlsx
- **Isi**: Data lengkap residual risk dalam spreadsheet
- **Kegunaan**: Analisis lanjutan, sharing data, backup

### 2. PDF Report
- **Format**: File .pdf
- **Isi**: Laporan profesional dengan statistik dan tabel
- **Kegunaan**: Presentasi, dokumentasi formal, arsip

### 3. Matrix Chart (PNG)
- **Format**: File .png
- **Isi**: Gambar residual risk matrix
- **Kegunaan**: Presentasi, dokumentasi visual

### 4. Comparison Chart (PNG)
- **Format**: File .png
- **Isi**: Gambar perbandingan inherent vs residual
- **Kegunaan**: Presentasi efektivitas penanganan risiko

### 5. Print Report
- **Format**: Print langsung
- **Isi**: Halaman lengkap dengan semua data
- **Kegunaan**: Hard copy untuk meeting, arsip fisik

## Tips Penggunaan

### 1. Analisis Efektivitas Penanganan Risiko
- Lihat kolom "Reduction" untuk melihat persentase pengurangan risiko
- Risiko dengan reduction tinggi menunjukkan penanganan yang efektif
- Risiko dengan reduction rendah perlu evaluasi strategi penanganan

### 2. Monitoring Risiko Tinggi
- Fokus pada risiko dengan level "High Risk" atau "Extreme High"
- Perhatikan tanggal "Next Review" untuk jadwal evaluasi
- Gunakan filter untuk melihat risiko per unit kerja

### 3. Pelaporan Berkala
- Download PDF report untuk laporan bulanan/triwulanan
- Gunakan Excel untuk analisis trend dan perhitungan lanjutan
- Simpan chart PNG untuk presentasi ke manajemen

### 4. Kolaborasi Tim
- Share Excel file dengan tim untuk review bersama
- Print report untuk meeting offline
- Gunakan filter untuk fokus pada area tanggung jawab masing-masing

## Troubleshooting

### Data Tidak Muncul
1. Pastikan sudah login dengan akun yang benar
2. Cek koneksi internet
3. Refresh halaman dengan tombol "Refresh Data"
4. Hubungi administrator jika masalah berlanjut

### Download Gagal
1. Pastikan browser mengizinkan download
2. Cek ruang penyimpanan device
3. Coba format download lain
4. Refresh halaman dan coba lagi

### Chart Tidak Tampil
1. Pastikan JavaScript diaktifkan di browser
2. Refresh halaman
3. Coba browser lain jika masalah berlanjut

## Keamanan Data

- Semua data dilindungi dengan authentication
- Akses dibatasi berdasarkan role dan organisasi
- Download file menggunakan token keamanan
- Data tidak dapat diakses tanpa login yang valid

## Dukungan

Jika mengalami masalah atau butuh bantuan:
1. Cek panduan ini terlebih dahulu
2. Hubungi administrator sistem
3. Laporkan bug atau saran perbaikan

---

**Catatan**: Pastikan selalu menggunakan data terbaru dengan melakukan refresh secara berkala.
